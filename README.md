# Snake Game

This is a basic Snake game implemented using HTML, CSS, and JavaScript.
![gameplay](/img/gameplay.png)
## How to Play

Enter in this website to play the game: [Snake Game](https://your-website.com)

- Score counter
- 'TAB' to restart the game
